The kid-simple version (5th grade)
Think of SDD like an empty LEGO table.
Spec Kit puts instruction cards on the table so Cursor can help you build step-by-step.

What you do
1) Open Cursor
2) Open the folder Desktop/SDD
(Cursor → File → Open Folder… → choose SDD)
Now Cursor is “inside” the project.

3) In Cursor chat, type these in order:
Make the rules for your project (how you want things built):

/speckit-constitution
Describe what you want to build (no tech details yet):

/speckit-specify
Say what tools/tech you want to use (React, Python, etc.):

/speckit-plan
Make a todo list (tasks the agent can follow):

/speckit-tasks
Build it:

/speckit-implement
GitHub part (optional, later)
If you want the tasks to appear as GitHub Issues:

First run in Terminal once:
gh auth login
Then in Cursor chat:

/speckit-taskstoissues
One sentence summary
Create/open the SDD folder in Cursor, then run the /speckit-* commands in chat in that order.
